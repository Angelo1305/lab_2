Course Catalog Description

This course covers the creation of web pages and sites using HTML, CSS, JavaScript, and graphical applications. Networked multimedia distribution technologies are also explored. The design of effective websites, including page layout, user interface design, graphic design, content flow, and site structure, as well as management of website resources including intranet management and design considerations, are addressed. Students design and create a major website with multiple pages and cross-linked structures.

Prerequisites: None

Credit: 2-2-3 Semester Hours

Course Outcomes

  • Recognize HTML, CSS, and JavaScript markup and code.
  • Select the proper markup tags or code to achieve a particular result.
  • Identify improperly used markup and code.
  • Produce modern standards-compliant web pages.
  • Deploy web pages to a GitHub server.

Course Objectives

  1. Explain the client and server architecture of both the internet and related web technologies.
  2. Use a basic text editor and other software tools to create web pages written in HTML, CSS, JavaScript, and jQuery.
  3. Deploy web pages to GitHub using version control and GitHub Desktop.
